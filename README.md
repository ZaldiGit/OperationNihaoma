# Nihaoma Student Operations System

MVP Streamlit untuk 3 modul inti:
- Calon Mahasiswa
- Dokumen
- Invoice & Pembayaran

## Fitur utama
- Dashboard operasional
- Form input calon mahasiswa
- Update progress & log
- Upload dan download dokumen
- Checklist dokumen wajib
- Buat invoice
- Catat pembayaran
- Download PDF invoice
- Backup / restore full package (.zip)

## Cara pakai
1. Jalankan app di Streamlit
2. Pilih **Muat Template Kosong** atau import backup
3. Tambahkan data mahasiswa
4. Upload dokumen
5. Buat invoice
6. Backup data via **Download full package backup (.zip)**

## Catatan
Versi ini dirancang agar bisa jalan di Streamlit tanpa backend tambahan.
Untuk persistensi dokumen, gunakan fitur backup package (.zip).
